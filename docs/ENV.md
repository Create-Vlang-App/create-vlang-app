# Environment variables (`CVA_*`)

| Var | Purpose |
|-----|---------|
| `CVA_CACHE_DIR` | Override cache root |
| `CVA_REFRESH` | `always\|stale\|manual` |
| `CVA_REFRESH_AFTER_HOURS` | Stale threshold |
| `CVA_OFFLINE` | Skip network |
| `CVA_NO_CATALOG_CACHE` | Bypass catalog cache |
| `CVA_STRICT_VERSION` | Strict version / install checks |
| `CVA_STRICT_REPRO` | Require full SHA refs |
| `CVA_SKIP_GIT` | Skip `git init` |

| `CVA_CATALOG_URL` | Override default templates.json URL |
